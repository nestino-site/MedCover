# MedCover

Travel health insurance comparison platform — SEO-focused frontend for international medical coverage guides.

## Structure

- `frontend/` — Next.js application (App Router, Tailwind CSS)
- `Docs/` — SEO architecture and content strategy documentation

## Getting Started

```bash
cd frontend
cp .env.example .env.local   # configure API keys and site URL
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Environment

See `frontend/.env.example` for required variables (backend API, site credentials, revalidation webhook).
